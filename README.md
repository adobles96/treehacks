## Setup
1. install all npm dependencies
```bash
cd emissions/static
npm install
```
2. source into the virtual environment
```bash
source emissions/py_venv/bin/activate
```

3. run the flask server
```bash
flask run
```

4. open localhost:5000 in your browser
